OVERVIEW

Description

The mhash library provides an easy way to access strong hashes such as MD5, SHA1 and other algorithms.

Features

The mhash library provides the following features:

o Free software (GNU Lesser General Public License) o Reentrancy support o Support for known strong algorithms o Support for less-known algorithms such as GOST o Should work on every ANSI C conforming platform
